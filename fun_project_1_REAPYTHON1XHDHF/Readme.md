TUTORIAL Cara Penggunaan App TES KEPRIBADIAN :
1. Ketik di terminal "streamlit run app.py"
2. Masukkan nama kamu terlebih dahulu
3. Jawab semua (6) pertanyaan dengan sesuai preferensi kamu
4. Hasil akan terlihat jika kamu menekan tombol Lihat Hasil
    a). Jika nilai di 2 atau 3 kepribadian sama rata, maka kamu akan diminta untuk melakukan tes lagi
    b). Jika kamu condong ke kepribadian Introvert maka akan ada GIF dan penjelasan singkat mengenai kepribadian Introvert 
    c). Jika kamu condong ke kepribadian Ekstrovert maka akan ada GIF dan penjelasan singkat mengenai kepribadian Ekstrovert
    d). Jika kamu condong ke kepribadian Ambivert maka akan ada GIF dan penjelasan singkat mengenai kepribadian Ambivert
