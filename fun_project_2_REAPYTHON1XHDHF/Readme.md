Tutorial Penggunaan Chatbot : 
1. Install terlebih dahulu library yang ada di requirements.txt\n
    Jalankan perintah pip install -r requirements.txt
2. Run aplikasi chatbot\n
    Jalankan perintah atreamlit run app.py
3. Silakan bertanya apapun. Sama seperti halnya pada ChatGPT, kita bisa mengupload dokumen dan bertanya seputar dokumen tersebut\n
    Dokumen yang bisa diupload harus berformat .docx atau .pdf
